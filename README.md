# Webpack Typescript Starter

### 使用方式

1. 克隆项目到本地
   ```shell
   git clone https://github.com/hungtcs-lab/three-js-demo.git your-project-name
   ```
2. 切换到项目目录
   ```shell
   cd your-project-name
   ```
3. 删除远程分支，或者切换为自己的仓库远程分支
   ```shell
   git remote remove origin
   ```
4. 安装 npm 依赖，接着启动项目
   ```shell
   npm install
   npm start
   ```
5. 使用浏览器访问，默认端口为`3100`
